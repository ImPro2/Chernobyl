# Chernobyl Engine
A game engine written in C++

## Cloning
Clone the project by typing `git clone --recursive https://github.com/ImPro2/Chernobyl`.
Go into the scripts folder and run `WinGenprojects.bat`.
Open the solution file in the root directory and build the solution.
